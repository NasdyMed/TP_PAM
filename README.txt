Nous avons créé notre propre JSON, et on l'a hebergé sur le site https://jsonkeeper.com.
Notre JSON contient Nom,Description,rating,image,lien(on l'a pas utilisé au final).

L'application utilise la librairy Volley pour faire la requête GET sur notre JSON.
On utilise aussi la librairy Glid pour la gestion d'image.


Notre application est une liste de film, l'utilisateur peut sélectionner un film pour savoir plus de detail.
